## 1. 클로드 코드 (Claude Code)
터미널 기반의 AI 개발 에이전트로, 파일 접근, 명령 실행, 코드 수정 및 테스트 자동화를 직접 수행합니다.

### 🛠️ 설치 및 실행 (Setup)
* **사전 요구사항:** Node.js 18 이상이 설치되어 있어야 합니다.
* **설치 명령어:** 터미널에서 아래 명령어를 입력합니다.
    ```bash
    npm install -g @anthropic-ai/claude-code
    ```

    ```bash
    claude auth
    ```

* **실행:** 프로젝트 루트 폴더에서 다음을 입력하여 실행합니다.
    ```bash
    claude
    ```

### 🛡️ 권한 제어 및 위험 모드 (Danger Zone)

## 2. 커서 AI (Cursor AI)
VS Code를 기반으로 제작된 AI 전용 코드 편집기로, 시각적인 코드 작성과 실시간 협업에 최적화되어 있습니다.

### ⌨️ 핵심 단축키 및 기능
| 단축키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **`Cmd(Ctrl) + K`** | **Inline Edit** | 코드 영역을 선택하고 직접 수정을 지시합니다. |
| **`Cmd(Ctrl) + L`** | **Chat** | 프로젝트 전체나 특정 파일을 대상으로 질문을 주고받습니다. |
| **`Cmd(Ctrl) + I`** | **Composer** | 여러 파일에 걸친 변경 사항을 한꺼번에 생성/수정합니다. (가장 강력) |
| **`Cmd(Ctrl) + J`** | **Terminal Chat** | 터미널에 발생한 에러를 즉시 분석하여 해결책을 제시합니다. |

### 🔍 컨텍스트 활용 (Symbols)
채팅창에서 `@` 기호를 사용하여 AI에게 명확한 맥락을 제공합니다.
* **`@Files`**: 특정 파일이나 폴더를 참조합니다.
* **`@Codebase`**: 프로젝트 전체 코드를 분석 대상으로 지정합니다.
* **`@Web`**: 최신 기술 문서나 스택 오버플로우 정보를 검색합니다.
* **`.cursorrules`**: 프로젝트 루트에 이 파일을 생성하여 AI가 지켜야 할 코딩 스타일(예: "모든 함수는 JSDoc 포함")을 강제합니다.

### 💡 추천 프롬프트
> **(Composer 모드에서)**: **"@Codebase 현재 데이터 모델을 기반으로 새로운 분석 대시보드 컴포넌트를 만들어줘. UI는 Tailwind CSS를 사용하고, 데이터 검증 로직은 기존 `ValidationService`를 상속받아 구현해."**
